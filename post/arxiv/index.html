My checklist for uploading papers to ArXiV

Apr 21, 2019 2 min read

Setup

  1. Define arxiv_generic/ folder and copy over the relevant files.
  2. Get the generic template for arxiv from https://github.com/kourgeorge/arxiv-style.
    1. Copy over the arxiv.sty into the arxiv_generic/ folder.
    2. You can get the sty file directly by using the link https://github.com/kourgeorge/arxiv-style/raw/master/arxiv.sty.
  3. Copy over the tex files into the arxiv_generic/ folder. Rename the main file as main.tex.

Tex edits

  1. Preamble: Instead of \documentclass[twocolumn]{autart} or any other conference/journal specific commands, use the following preamble 
    \documentclass{article}    %
\usepackage{arxiv}
   
\usepackage{bookmark}
\usepackage{amsthm,amssymb,amsmath}
\newtheorem{thm}{Theorem}
\newtheorem{cor}[thm]{Corollary}
\newtheorem{lem}[thm]{Lemma}
\newtheorem{assum}[thm]{Assumption}
\newtheorem{prop}[thm]{Proposition}
\theoremstyle{definition}
\newtheorem{defn}[thm]{Definition}
\newtheorem{exmp}[thm]{Example}
\newtheorem{rem}[thm]{Remark}
\newtheorem{prob}[thm]{Problem}
\newtheorem{alg}{Algorithm}
  2. Document heading 
    \title{TITLE OF THE PAPER}
\author{Author 1 \\
    Address \\
    \texttt{Email}\\
    \And
    Author 2 \\
    Address \\
    \texttt{Email}\\
    \thanks{Foot information}
}
\date{}
\renewcommand{\headeright}{}    % To remove the Preprint
\renewcommand{\undertitle}{}    % To remove the Preprint
   
\begin{document}
\maketitle
  3. Keyword 
    \begin{keyword}                         
    Stochastic reachability;
    chance constrained optimization;
    convex optimization;
    stochastic optimal control
\end{keyword}                            
% changes to
\keywords{Stochastic reachability \and chance constrained
    optimization \and convex optimization \and stochastic
optimal control}
  4. Bibliography
    1. Copy over original_tex_file.bbl into the arxiv folder. Avoid using the same file name as the main.tex to avoid latexmk from overwriting it.
    2. Add \input{original_tex_file.bbl} to the end of the file instead of the bibliography commands.

Other considerations

  1. Figures:
    1. Make sure figures are placed correctly. Sometimes figures that were forced to below each other can now be placed side-by-side.
    2. Remove all .fig and .svg files
    3. Remove any extraneous files.
  2. Make sure equations are single lined, when possible, to reduce the page length.
  3. Use \adjustbox{width=DESIRED_WIDTH}{BLAHBLAH} to resize tables to the desired shape.

Cleaning up the files for arXiv upload

  1. Clone the git repository https://github.com/google-research/arxiv-latex-cleaner
  2. Do python setup.py install to make it like a command line`
  3. Run arxiv_latex_cleaner /path/to/arxiv_generic to generate a folder /path/to/arxiv_generic_arXiv.
  4. You may have to copy over all the figures again, since arxiv_latex_cleaner may do some unnecessary image manipulation.
  5. Remove the .fls and .fdb_latexmk files from the arxiv_generic_arXiv folder.
  6. Zip the folder arxiv_generic_arXiv and upload.
